Navrhujeme a implementujeme CI/CD pipeline, Infrastructure as Code a cloudové pracovné postupy, ktoré urýchľujú vaše dodávky pri zachovaní spoľahlivosti — v cloude aj na vlastnej infraštruktúre.

## Čo ponúkame

- Návrh a implementácia CI/CD pipeline (GitLab CI, GitHub Actions, Jenkins)
- Infrastructure as Code s Ansible a Terraform
- Orchestrácia kontajnerov s Kubernetes a Docker
- Nasadenie a aktualizácia Kubernetes na vlastnej infraštruktúre (Kubespray, kubeadm)
- Cloudová architektúra AWS a migrácia
- GitOps pracovné postupy a automatizácia nasadenia
